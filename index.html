How to use this mirror (and what your GTM tag must do)
  1. The page auto-loads container GTM-TTC6MVJ. Change it in the top bar or with ?gtm=GTM-XXXXXXX.
  2. Because this page does not push to the data layer, your GTM Custom HTML tag (fired on All Pages) must listen for Elementor's submit_success and push the event with the email. Critically, read the email at submit time — the form resets on success, so reading it after submit_success returns empty.
  3. Then a Custom Event trigger on your pushed event name fires the Zeotap Collect tag (Login Event = that event, Capture Email mapped to a Data Layer Variable).
  4. Enter GTM Preview, point it at this page, submit the form, and watch: the lifecycle log shows submit_success firing, your pushed event appears in the monitor with the email, the Zeotap SDK goes green, and DevTools → Network shows the content.zeotap.com call.
