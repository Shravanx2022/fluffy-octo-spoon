How to use this page
  1. The page auto-loads container GTM-TTC6MVJ. To use a different one, type it above and press Load container (reloads with ?gtm=).
  2. In that GTM workspace, add the Zeotap Collect Tag with your own write key, then attach a Custom Event trigger on form_submitted and map a Data Layer Variable email into Capture Email. (Full steps are in the setup guide.)
  3. Enter Preview in GTM, point Tag Assistant at this page's URL, submit the form, and watch: the event appears in the monitor, Zeotap SDK flips to detected, and in DevTools → Network you'll see the call to spl.zeotap.com / content.zeotap.com.

Tip: append ?gtm=GTM-XXXXXXX to the URL to preset the container without typing.
